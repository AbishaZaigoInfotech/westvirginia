Product Creation


Method: POST
Field * required Validations Rule Testing Inputs
URL http://devapigateway-saas.zaicrm.com/z1/accounting/inventory
_token * Bearer Token: "token" Pass it in header as all other request
item_type Optional it Should be product OR general. general - is used to create general products product
name * Manadatory maximum 190 Characters accepted. No. 276
description Nullable It may be long text. Ac Installtion
sku Optional Maximum 45 character accepted 18462832
status Optional Should be active or inactive active
measurement_unit Optional ge from unit api need to pass unit id 1
category Optional get from category api Need to pass category id 1
brand_name Optional Maximum 45 character accepted, If you provide brand id it will map it to the brand. if you are type new brand name it will create new brand and map it to the product. Apple
price Optional it should be numeric. 10000
current_stock Optional should be numeric. 100
low_stock Optional should be numeric
is_taxable Optional should be true or false. If you provide true you can pass tax id we can map it to the tax. If you are provide a new tax name we can create new one and map it the product 2
tax Optional, If is_taxable is true it is Mandatory otherwise optional should be numeric and get from tax api OR You can create new tax name and percentage 2
tax_percentage Optional, If is_taxable is true it is Mandatory otherwise optional should be numeric 2
images Optional [ it should be a valid format(pdf, jpeg, png, docx) and the file size should be 30Mb , mimes:pdf, jpeg, png, docx,'max:30480'] filename
Note: Images can be uplaod mutiple file by following format images[0],images[1],images[2]. Upload Maximum 10 files only.
